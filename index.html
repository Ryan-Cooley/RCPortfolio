TIP3P Water Box Simulation

Molecular Dynamics

Project Overview

The TIP3P water box simulation is a molecular dynamics model that simulates the behavior and movement of water molecules using the TIP3P (Transferable Intermolecular Potential with 3 Points) model. This model is widely used in computational chemistry because of its efficiency and reasonable accuracy in representing water's physical properties.

In this simulation, the TIP3P water box is visualized using VMD (Visual Molecular Dynamics), demonstrating how water molecules interact in a confined space over time, providing insights into their behavior under different conditions.

Technical Details

The files were generated using the OpenMM toolkit, a versatile platform for running molecular simulations, which provides the tools necessary to build, parameterize, and simulate molecular systems efficiently.

File Components:

  • INPCRD File (Input Coordinates): This file contains the initial coordinates of all atoms in the system, defining the starting configuration for the simulation.
  • PRMTOP File (Parameter Topology): This file describes the molecular structure, including atomic properties, bonding information, and parameters needed to calculate forces and energies throughout the simulation. It serves as a blueprint for the molecular interactions.
  • Trajectory File (DCD): The trajectory file (`.dcd`) records the positions of all atoms over time during the simulation, allowing for visualization and analysis of how the system evolves.

Snake Game

Python Programming

Project Overview

A simple Python game where the player controls a snake to collect food, growing longer with each item collected. Avoid hitting the walls or yourself!

This project was inspired by Angela Yu's 100 Days of Code: The Complete Python Pro Bootcamp and demonstrates fundamental programming concepts including:

  • Object-Oriented Programming: Using classes to organize game logic
  • Event Handling: Processing keyboard input for movement
  • Game Loop: Managing game state and rendering
  • Collision Detection: Checking for wall and self collisions
  • Score Tracking: Keeping track of food collected

Two Versions Available

I've created two versions of the Snake game to demonstrate different programming approaches:

Basic Version (snake.py)

A straightforward implementation using basic Python concepts and the turtle graphics library. This version is perfect for understanding the core game mechanics.

Class-Based Version (snake_game_classes.py)

A more advanced implementation using object-oriented programming principles. This version demonstrates better code organization and reusability.

How to Run

To run either version of the game:

  1. Download the Python file
  2. Make sure you have Python installed on your system
  3. Open a terminal/command prompt
  4. Navigate to the directory containing the file
  5. Run: python snake.py or python snake_game_classes.py
