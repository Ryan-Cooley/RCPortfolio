TIP3P Water Box Simulation

Molecular Dynamics

Project Overview

This simulation models the behavior of water molecules using the TIP3P (Transferable Intermolecular Potential with 3 Points) model, which is widely used in computational chemistry due to its efficiency and reasonable accuracy in representing water’s physical properties.

The TIP3P water box is visualized using VMD (Visual Molecular Dynamics), showcasing how water molecules interact within a confined space over time. This provides insight into their behavior under realistic, periodic conditions.

Technical Details

The simulation was built and run using the OpenMM toolkit, a powerful engine for molecular simulations. System setup was performed using AmberTools, and custom Python scripts were developed for analysis and visualization.

Simulation Components:

  • Water Box Setup (Leap): AmberTools script to generate a TIP3P water box with periodic boundary conditions
  • Molecular Dynamics Simulation: OpenMM Python script with energy minimization, equilibration, and production phases
  • Distance Analysis: MDTraj-based Python script to compute nearest-neighbor distances between water oxygens
  • Periodic Boundary Analysis: Extended distance analysis accounting for periodic boundary conditions
  • Data Visualization: Python script to generate histograms and plots using NumPy and Matplotlib

Analysis Results

Histogram of nearest-neighbor distances between water oxygens, illustrating the molecular spacing and statistical distribution.

Downloads

Molecular Dynamics Simulation Water Box Setup (Leap) Distance Analysis Periodic Boundary Analysis Data Visualization

How to Run

To run the molecular dynamics simulation:

  1. Download all Python scripts and the Leap file
  2. Install required packages: pip install openmm parmed numpy matplotlib mdtraj
  3. Set up the water box: tleap -f TIP3P_Water_Sim/tip3p_water_box_setup.leap
  4. Run the simulation: python TIP3P_Water_Sim/tip3p_molecular_dynamics_simulation.py
  5. Analyze results: python TIP3P_Water_Sim/tip3p_distance_analysis.py and python TIP3P_Water_Sim/tip3p_periodic_boundary_analysis.py
  6. Generate visualizations: python TIP3P_Water_Sim/tip3p_data_visualization.py
  7. Visualize trajectory using VMD or similar software

Simulation Details

  • Temperature: 300 K (room temperature)
  • Integration: Langevin dynamics with 1/ps friction
  • Time Step: 0.002 ps (2 femtoseconds)
  • Equilibration: 100 ps (50,000 steps)
  • Production: 20,000 ps (10,000,000 steps)
  • Periodic Boundary Conditions: PME (Particle Mesh Ewald) method
  • Non-bonded Cutoff: 1.0 nanometer

Note: This project showcases modern scientific computing, including molecular simulation, data analysis, and statistical visualization — all conducted using open-source tools like OpenMM, AmberTools, MDTraj, and Matplotlib.

Snake Game

Python Programming

Project Overview

A classic Snake game implemented in Python with the turtle graphics library. Guide your snake to eat food, grow longer, and rack up points—while steering clear of walls and your own tail!

Inspired by Angela Yu’s 100 Days of Code: The Complete Python Pro Bootcamp, this project showcases key programming principles:

  • Object-Oriented Design: Encapsulate game elements in reusable classes
  • Event Handling: Capture and respond to keyboard input for smooth movement
  • Main Game Loop: Continuously update and render game state
  • Collision Detection: Detect and handle collisions with walls or the snake itself
  • Score Management: Track and display your current score in real time

Project Structure

The codebase is split into two modules for clarity and maintainability:

Main Game File (snake.py)

Entry point to launch the game, containing:

  • Setup of the game window and turtle environment
  • The primary game loop for movement, rendering, and timing
  • Collision checks and game-over logic
  • High-score display and reset handling

Classes Module (snake_game_classes.py)

Defines the core game objects:

  • Snake: Manages body segments, directional movement, and self-collision checks
  • Food: Randomly places food items on the screen and handles respawning
  • ScoreBoard: Keeps track of and renders the player’s score at the top of the screen

Downloads

Download Main Game File Download Classes Module

How to Run

To start playing:

  1. Download both Snake_Game/snake.py and Snake_Game/snake_game_classes.py
  2. Ensure you have Python 3 installed
  3. Open a terminal or command prompt
  4. Navigate to the folder containing the files
  5. Execute: python Snake_Game/snake.py

Note: Both files are required—Snake_Game/snake.py imports classes from Snake_Game/snake_game_classes.py.

Disclaimer: This game is a simple educational project and may have some quirks—such as food sometimes appearing near the edge, or the score being displayed at the top. For best results, run it in a standard Python environment with the turtle module installed.

PCEP Certification

Python Certification

Certification Overview

The PCEP (Certified Entry-Level Python Programmer) certification is a professional credential that validates fundamental Python programming skills. This certification demonstrates proficiency in Python syntax, data types, control structures, and basic programming concepts.

What PCEP Covers

The PCEP certification validates knowledge in:

  • Python Fundamentals: Variables, data types, operators, and expressions
  • Control Structures: Conditional statements, loops, and flow control
  • Data Collections: Lists, tuples, dictionaries, and sets
  • Functions: Function definition, parameters, and return values
  • Exception Handling: Basic error handling and debugging
  • File Operations: Reading and writing files in Python

Why This Matters

Earning the PCEP certification demonstrates:

  • Solid foundation in Python programming
  • Commitment to professional development
  • Industry-recognized programming skills
  • Readiness for more advanced Python certifications

This certification serves as a stepping stone toward more advanced Python certifications like PCAP (Certified Associate in Python Programming) and PCPP (Certified Professional in Python Programming).
