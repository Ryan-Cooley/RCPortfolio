TIP3P Water Box Simulation

View Code

Python • Computational Chemistry & Systems Analysis

Why This Project?

I wanted to understand how water molecules self-organize under realistic conditions and build a reusable workflow for analyzing molecular systems. By combining OpenMM’s simulation engine with custom Python analysis scripts, this project demonstrates end-to-end systems thinking—from model setup and execution to automated data processing and visualization.

Project Overview

TIP3P Water Box Simulation uses AmberTools to generate a cubic box of TIP3P water molecules, runs dynamics via OpenMM, and employs MDTraj and Matplotlib for statistical analysis and plots. The codebase is modular, fully reproducible, and container-ready for easy sharing and extension.

Key Advantages

  • Accurate & Efficient: Leverages the TIP3P model and Particle Mesh Ewald to balance computational cost and physical fidelity.
  • End-to-End Automation: Scripts automate box setup, simulation, analysis, and figure generation—eliminating manual steps.
  • Systems Insight: Nearest‑neighbor distance histograms and trajectory visuals reveal molecular packing and dynamic behavior.
  • Reproducibility: Configuration files, environment specs, and containerization ensure others can reproduce results exactly.
  • Extensible Framework: Easily swap in different force fields or analysis metrics for broader molecular-system studies.

Skills Demonstrated

  • Setting up and running molecular dynamics with OpenMM & AmberTools
  • Writing Python scripts for data extraction, statistical analysis, and plotting
  • Applying systems-level thinking to automate complex workflows
  • Ensuring reproducibility via environment files and containerization
  • Visual storytelling through interactive GIFs and publication-ready plots

Quickstart

  1. Clone the repo: git clone https://github.com/Ryan-Cooley/TIP3P-Water-Simulation.git
  2. Install dependencies: pip install -r requirements.txt (or use environment.yml)
  3. Generate the water box: tleap -f tip3p_water_box_setup.leap
  4. Run dynamics: python tip3p_molecular_dynamics_simulation.py
  5. Analyze distances: python tip3p_distance_analysis.py
  6. Visualize results: python tip3p_data_visualization.py
  7. View trajectory: open trajectory.gif or load the DCD in VMD

Sample Outputs

Molecular Spacing: Histogram of nearest‑neighbor oxygen–oxygen distances, showing the characteristic 0.28 nm peak of liquid water.

Why It Matters

  • Demonstrates scalable molecular dynamics workflows for computational chemistry
  • Highlights systems engineering: from input preparation through automated analysis
  • Provides a template for reliable, reproducible scientific research using open‑source tools

Snake Game

Python Programming

Project Overview

A classic Snake game implemented in Python with the turtle graphics library. Guide your snake to eat food, grow longer, and rack up points—while steering clear of walls and your own tail!

Project Structure

The codebase is split into two modules for clarity and maintainability:

Main Game File (snake.py)

Entry point to launch the game, containing:

  • Setup of the game window and turtle environment
  • The primary game loop for movement, rendering, and timing
  • Collision checks and game-over logic
  • High-score display and reset handling

Classes Module (snake_game_classes.py)

Defines the core game objects:

  • Snake: Manages body segments, directional movement, and self-collision checks
  • Food: Randomly places food items on the screen and handles respawning
  • ScoreBoard: Keeps track of and renders the player’s score at the top of the screen

Downloads

Download Main Game File Download Classes Module

How to Run

  1. Download both Snake_Game/snake.py and Snake_Game/snake_game_classes.py
  2. Ensure you have Python 3 installed
  3. Open a terminal or command prompt
  4. Navigate to the folder containing the files
  5. Execute: python Snake_Game/snake.py

Note: Both files are required—Snake_Game/snake.py imports classes from Snake_Game/snake_game_classes.py.

Disclaimer: This game is a simple educational project and may have some quirks—such as food sometimes appearing near the edge, or the score being displayed at the top. For best results, run it in a standard Python environment with the turtle module installed.

PCEP Certification

Python Certification

Certification Overview

The PCEP (Certified Entry-Level Python Programmer) certification is a professional credential that validates fundamental Python programming skills. This certification demonstrates proficiency in Python syntax, data types, control structures, and basic programming concepts.

What PCEP Covers

The PCEP certification validates knowledge in:

  • Python Fundamentals: Variables, data types, operators, and expressions
  • Control Structures: Conditional statements, loops, and flow control
  • Data Collections: Lists, tuples, dictionaries, and sets
  • Functions: Function definition, parameters, and return values
  • Exception Handling: Basic error handling and debugging
  • File Operations: Reading and writing files in Python

Why This Matters

Earning the PCEP certification demonstrates:

  • Solid foundation in Python programming
  • Commitment to professional development
  • Industry-recognized programming skills
  • Readiness for more advanced Python certifications

This certification serves as a stepping stone toward more advanced Python certifications like PCAP (Certified Associate in Python Programming) and PCPP (Certified Professional in Python Programming).
