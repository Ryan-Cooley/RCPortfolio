Snake Game

Python Programming

Project Overview

A simple Python game where the player controls a snake to collect food, growing longer with each item collected. Avoid hitting the walls or yourself!

This project was inspired by Angela Yu's 100 Days of Code: The Complete Python Pro Bootcamp and demonstrates fundamental programming concepts including:

  • Object-Oriented Programming: Using classes to organize game logic
  • Event Handling: Processing keyboard input for movement
  • Game Loop: Managing game state and rendering
  • Collision Detection: Checking for wall and self collisions
  • Score Tracking: Keeping track of food collected

Two Versions Available

I've created two versions of the Snake game to demonstrate different programming approaches:

Basic Version (snake.py)

A straightforward implementation using basic Python concepts and the turtle graphics library. This version is perfect for understanding the core game mechanics.

Class-Based Version (snake_game_classes.py)

A more advanced implementation using object-oriented programming principles. This version demonstrates better code organization and reusability.

How to Run

To run either version of the game:

  1. Download the Python file
  2. Make sure you have Python installed on your system
  3. Open a terminal/command prompt
  4. Navigate to the directory containing the file
  5. Run: python snake.py or python snake_game_classes.py
